## 五、实施决策建议

1. **G01 结构化蓝图**：这是最大的用户体验差距。建议将 8 个步骤按 PRD §8.3 的字段规格逐一实现
2. **G02 Frontmatter**：小改动，应优先修复
3. **G03 迁移文件**：对开发流程有长期价值，建议抽取
4. **G06 shadcn/ui**：**不建议**在 MVP 阶段替换，自定义 Tailwind 方案完全可用
5. **G09 独立 Agent**：**不建议**实现，当前 AiService + PromptBuilder 架构更简洁
