# 生成章节草稿

## 目标

输出可直接入稿的章节正文，而非“需要再加工的大纲体文本”。

## 章节执行流程

1. 明确本章功能：推进主线/角色深化/伏笔种植/伏笔回收/情绪缓冲。
2. 拆场景：3-6个场景，每个场景只承担一个核心变化。
3. 编排节奏：张弛交替，避免全章同速。
4. 收束钩子：结尾留下可驱动下一章的悬念或决策。

## 类型差异化指南

### 玄幻/仙侠

1. 核心：修炼规则、力量层级、代价。
2. 场景重点：战斗中的规则兑现，不靠“临场开挂”。
3. 常见错：境界描述混乱、术语漂移。

### 都市/现实

1. 核心：人物关系与社会约束。
2. 场景重点：对话推进和现实细节（地点、职业、利益）。
3. 常见错：冲突戏剧化过度，脱离现实逻辑。

### 科幻

1. 核心：技术边界与社会后果。
2. 场景重点：技术如何改变选择，不只展示名词。
3. 常见错：设定炫技，人物行动动机变弱。

### 悬疑/推理

1. 核心：线索分布、误导、公平揭秘。
2. 场景重点：每个场景至少投放或校正一个线索。
3. 常见错：线索集中结尾，前文信息价值不足。

### 历史/架空历史

1. 核心：时代语境、制度约束、礼法细节。
2. 场景重点：冲突与行动必须受时代条件限制。
3. 常见错：现代思维直接套进古代场景。

### 同人

1. 核心：角色原著一致性 + 新冲突有效增量。
2. 场景重点：保留人物“声纹”和行为底层逻辑。
3. 常见错：角色OOC（性格跑偏）或纯复述原作桥段。

## 章节开头钩子模板（至少选一种）

1. 动作钩子模板：
"[角色]刚完成[高风险动作]，却发现[意外后果]。"
2. 对话钩子模板：
"“[一句带冲突的信息]。”[角色]说完，[对方]的反应立刻改变了局势。"
3. 反常钩子模板：
"按理说[常态]，但今天[反常现象]先发生了。"
4. 倒计时钩子模板：
"距离[关键事件]只剩[时长]，而[关键条件]仍未满足。"

## 多视角切换规范

1. 一个场景只允许一个内心视角。
2. 视角切换必须有段落断点和场景触发。
3. 切换后首句需给新视角锚点（角色/地点/感知对象）。

常见错误：

1. 同段内连续写两人内心。
2. 为解释信息强行全知插话。
3. 切换后语气未变化，导致“同一声音”。

## 节奏控制方法

1. 快节奏段：短句+动作+低解释密度。
2. 中节奏段：对话+动作均衡。
3. 慢节奏段：心理/环境/关系沉淀。

节奏指标建议：

1. 连续2个高张力场景后，至少给1个缓冲场景。
2. 单场景信息新增点控制在1-3个。
3. 每400-600字至少发生一次状态变化。

## 结尾悬念设计手法

1. 信息悬念：揭示新事实但不揭示解释。
2. 决策悬念：角色必须在两难中做选择。
3. 后果悬念：本章行动引出更大代价。
4. 身份悬念：关键人物真实立场被撬开一角。

## 质量底线

1. 人物行为必须有动机支撑。
2. 规则不被临时破坏。
3. 场景之间存在可见因果连接。
4. 输出为连续正文，不含“生成说明”。
