项目背景
NovelForge — AI 驱动的长篇小说创作辅助软件。
核心开发理念：

方便每位小说创作者，都能使用这个软件快速创作出高质量、逻辑性强的长篇小说。

技术栈：

前端：React 19 + TypeScript + Vite + Zustand + Tauri API
后端：Rust 2021 + Tauri 2 + rusqlite + reqwest + tokio + keyring
存储：章节正文 Markdown 文件化 + SQLite 元数据

核心架构：

前端 UI 层：src/pages/* + src/app/router.tsx
前端调用层：src/api/*（统一通过 invokeCommand 调 Rust command）
后端命令层：src-tauri/src/commands/*（Tauri 命令门面）
后端服务层：src-tauri/src/services/*（业务核心）
基础设施层：src-tauri/src/infra/*（路径、数据库、迁移）
